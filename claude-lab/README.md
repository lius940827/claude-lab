# 劭廷 Studio — 個人品牌 Landing Page

平面設計師「劭廷」的個人作品集網站，展示品牌識別、視覺傳達與數位設計三大領域的精選專案，並附有聯繫區塊方便客戶快速取得聯繫。

## 功能

- **Hero 區塊**：全版主視覺，含動態漸層 Orb 背景與 Fade-up 動畫
- **精選作品（Works）**：三張作品卡片，各附 Tag、說明與技術 Chip
- **聯繫（Contact）**：一鍵 mailto 開啟郵件客戶端
- **響應式導覽列**：桌面版顯示連結，手機版自動收合
- **RWD 手機版優化**：`≤640px` 時按鈕改為全寬、卡片單欄排列、字級與間距重新調整，確保觸控友善

## 使用技術

| 技術 | 說明 |
|------|------|
| HTML5 | 語意化標籤結構 |
| CSS3 | 自訂 CSS 變數、Flexbox、CSS Grid、`clamp()`、Media Queries |
| Google Fonts | Inter + Noto Sans TC |
| CSS Animation | `@keyframes fadeUp / fadeIn / drift` |

> 無任何外部 JavaScript 依賴，純 HTML + CSS 實作。

## 如何開啟

1. 將整個專案資料夾下載或 clone 到本機：
   ```bash
   git clone https://github.com/lius940827/claude-lab.git
   ```
2. 進入專案目錄，找到 `claude-lab/claude-lab/index.html`
3. 直接用瀏覽器開啟該檔案（雙擊，或將路徑拖入瀏覽器網址列）
4. 即可在本機預覽完整網站，無需伺服器

> 建議使用 Chrome / Edge / Firefox 最新版以獲得最佳體驗。
